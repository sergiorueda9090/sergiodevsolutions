Guía Paso a Paso para Instalar Django

Paso 1: Instalar Django

Para instalar Django en tu entorno, primero asegúrate de tener pip instalado. Luego ejecuta el siguiente comando:

pip install django

Paso 2: Crear un Proyecto de Django

Una vez que Django esté instalado, puedes crear un nuevo proyecto ejecutando el siguiente comando:

django-admin startproject myproject

Paso 3: Ejecutar el Servidor de Desarrollo

Para verificar que todo está funcionando correctamente, navega al directorio del proyecto y ejecuta el siguiente comando:

python manage.py runserver
